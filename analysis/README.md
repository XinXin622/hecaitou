# 学习如何写推荐文章：和菜头风格解构

## 项目概述

本项目通过分析和菜头的30篇推荐文章，解构其写作风格与核心策略，提炼出可复用的写作模板。

## 文章分类

共30篇文章，分为三大类：

| 类型 | 占比 | 代表作品 |
|------|------|----------|
| **电影评论** | 40% | 《椒麻堂会》、《森气气》、《马丁易老，诺兰难封》 |
| **书籍推荐** | 40% | 《世界上最快乐的人》、《阅读入门》、《像个盗贼那样读书》 |
| **社会思辨** | 20% | 《肉都烂在锅里》、《拒绝新闻的生活》 |

## 核心发现

### 写作理念

> "你获得什么样的读者，你本身就是什么样的人"

### 风格要素

- **诚实 > 推销**：先承认障碍，再说价值
- **思辨 > 断言**：不绝对化，承认复杂性
- **启蒙 > 营销**：提供思维启发，而非简单推销
- **对话感 > 说教感**：自问自答，商量式语气

### 推荐文章框架

1. 开篇：个人体验切入 / 悖论引入 / 直接表态
2. 障碍声明：诚实告知门槛和可能的不适
3. 多维分析：从2-4个角度展开讨论
4. 深层价值：超越"好看/好读"，说明思维启发
5. 使用指南：如何进入、适合谁、怎么用
6. 价值升华：连接到更大的人生/社会议题

## 如何使用

1. 先阅读 `01-风格总结.md` 了解整体风格
2. 通过 `02-推荐策略.md` 掌握核心技巧
3. 精读 `03-世界上最快乐的人-精读.md` 看具体应用
4. 使用 `04-写作模板.md` 进行实践
5. 参考 `05-练习建议.md` 持续提升
